# Geerdev Portfolio

Portfolio personal y CV web de **Germán Fernández Sierra ("Geerdev")**, Full Stack Developer & DevOps. Sitio estático construido con Astro, React y Tailwind CSS. 👉 [geerdev.tech](https://geerdev.tech/)

## 🚀 Showcase de proyectos

Una selección de mis proyectos con su descripción, highlights y enlaces, para mostrar en qué he trabajado y con qué tecnologías.

## ✏️ Blog técnico

Artículos donde comparto lo que aprendo sobre desarrollo web, infraestructura, bases de datos, IA y más, filtrables por categoría.

## 📩 Contacta conmigo

¿Hablamos? Puedes escribirme por correo o encontrarme en mis redes (LinkedIn, GitHub) desde la cabecera del sitio o con la paleta de comandos (Cmd/Ctrl + K).

## 🖨️ Convertible en CV imprimible

El sitio está pensado para imprimirse como CV: pulsa Ctrl + P y obtén una versión limpia en PDF lista para enviar.
